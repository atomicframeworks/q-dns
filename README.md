QNS
=========

A small library that wraps Node.js DNS with Q promises.

## Installation

  npm install qns --save

## Usage

    var qns = require('qns');

	qns.lookup('www.google.com').then(function (results) {
		console.dir(results);
	});

## Tests

    npm test
